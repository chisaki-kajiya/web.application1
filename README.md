
# Todoアプリ
タスクの管理ができるTodoアプリ
![スクリーンショット 2022-09-25 17 43 43](https://user-images.githubusercontent.com/109803113/192135333-17978d15-0eab-4930-8c2b-91f895451dfb.png)

## 作成した目的
簡単にタスク管理をできるようにする為

## 機能一覧
- タスクの登録
- タスク名の変更および更新
- タスクの削除

## 使用技術（実行環境）
- Laravel　8.x
